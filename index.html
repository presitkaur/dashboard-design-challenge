About this Project

Tasked with creating an interactive dashboard for data visualisation, this project focuses on analysing weather patterns of over 500 cities. With data sourced from the OpenWeatherMap API, the analysis examines how weather is affected based on a city's proximity to the equator.

Graphs were created using Matplotlib, plotting various variables such as humidity or cloudiness against the latitude of a city. This project provides a thorough analysis of the findings as well as the source data used.
